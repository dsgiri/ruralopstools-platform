# User Flows

## Persona: Farmer / Rancher
1. **Discovery**: Lands on a specific tool subdomain (e.g., Tractor Logs) via SEO.
2. **Engagement**: Uses the free calculator/logger.
3. **Dashboard**: Navigates to the main RuralOpsTools hub to view all available tools.
4. **Retention**: Pins favorite tools to the sidebar for quick access.
