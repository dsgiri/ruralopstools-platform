# Product Requirements Document

## Project: RuralOpsTools

### Objective
To build niche digital tools and SEO landing pages for ruralopstools.com. The platform focuses on underserved operational software and digital tools for farms, ranches, homesteads, and remote rural infrastructure.

### Core Tools
- Soil Health
- Feed Calculator
- Agri-Weather
- Tractor Logs
- ...and many more field-specific utilities.

### UX Principles
- Practical, rugged, mobile-first design
- Outcome-driven, plainspoken copy
- Scannable in the field under sunlight
