# Design System & Style Guide

## Theme
- **Style**: Practical, rugged, field-operations-style design.
- **Framework**: Tailwind CSS 4.
- **Icons**: Lucide React.

## Colors (Current Palette)
- Backgrounds: Gray-50, White.
- Accents: Green, Amber, Blue, Orange, Teal (matching agricultural themes).
- Text: Gray-900 (primary), Gray-500 (secondary).

## Typography
- Font: Sans-serif (default Tailwind).
- Headers: Bold, clear, high contrast.
